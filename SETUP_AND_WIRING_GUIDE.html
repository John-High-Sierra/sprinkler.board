Section 2

Wiring to a 24VAC Sprinkler System

How a 24VAC System Works

Each zone valve is a solenoid that opens when 24VAC is applied across it. The controller completes the circuit for one zone at a time. All valves share a Common wire — one side of the 24VAC supply. The controller switches the other side (the hot leg) to each zone individually.

Figure 1 — 24VAC sprinkler system wiring. The Common wire (green) is shared by all valves and connects back to the transformer.

Wiring Steps

🔴
Turn off power at the breaker before wiringThe board uses mains voltage internally. All wiring must be done with power off.
  1. Identify your wires. At the old controller you will find a bundle of low-voltage wires. There will be one Common wire (usually white, labelled C or COM) and one wire per zone (usually coloured, labelled 1–8).
  2. Connect the Common wire to the COM terminal on the SprinKlr-8 board.
  3. Connect each zone wire to the matching relay terminal — Zone 1 wire to Z1, Zone 2 wire to Z2, and so on.
    If you have fewer than 8 zones, leave the unused relay terminals empty.
  4. Connect the 24VAC transformer. One leg of the transformer connects to the board's HOT input terminal. The other leg connects to the Common wire (or joins it at the valve end).
    If you are reusing an existing transformer from the old controller, the wiring is usually already in place — you only need to move the zone and common wires.
  5. Connect mains power (120 or 240VAC) to the L, N, and GND terminals on the board's onboard power supply. Restore power at the breaker.
💡
Tip — Test before closing upUse the Manual page in the web interface to activate each zone one at a time and confirm the correct valve opens before sealing the enclosure.
Section 3

Loading Firmware for the First Time

The board has no built-in USB port. Firmware is loaded once using a CP2102 USB-to-serial adapter. After the initial load, all future updates can be done wirelessly (see Section 7).

What You Need

Item
CP2102 USB serial adapterAny CP2102 or CH340 adapter will work
Arduino IDE 2.xFree download from arduino.cc
4 jumper wiresFemale-to-female

CP2102 Wiring

Locate the 6-pin header on the board near the IO0 and EN buttons.

Figure 2 — CP2102 to board wiring. Note TX and RX are crossed: adapter TX connects to board RX, and vice versa.

Arduino IDE Setup

  1. Install the ESP32 board package. In Arduino IDE go to File → Preferences and add this URL to Additional Boards Manager URLs:
    https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
  2. Go to Tools → Board Manager, search esp32, and install esp32 by Espressif Systems version 3.3.8.
  3. Install required libraries via Tools → Manage Libraries:
    LibraryAuthor
    WiFiManagertzapu
    ArduinoJsonbblanchon — install version 6.x
    NTPClientFabrice Weinberg
  4. Select board settings:
    SettingValue
    BoardESP32 Dev Module
    Partition SchemeDefault 4MB with spiffs
    Upload Speed921600
    PortWhichever COM port the CP2102 appears on

Flashing the Firmware

Figure 3 — Flash mode sequence. Must be done each time you upload via serial.
  1. Open sprinkler_controller.ino in Arduino IDE and click Upload.
  2. When the IDE shows "Connecting...", perform the boot sequence above: hold IO0 → tap EN → release IO0.
  3. The firmware uploads. When complete the Serial Monitor will show [BOOT] All systems GO.

Uploading the Web Interface

  1. In Arduino IDE press Ctrl+Shift+P and select Upload LittleFS image.
  2. Perform the same boot sequence: hold IO0 → tap EN → release IO0.
    Close the Serial Monitor first — it blocks the COM port.
  3. Wait for the upload to complete. The web interface is now stored on the board.
💡
Serial connection is only needed onceAfter the initial firmware and UI are loaded, all future updates can be done wirelessly from the web interface. The CP2102 adapter can be disconnected and stored.
Section 4

First Boot & WiFi Setup

  1. Power on the board. On first boot it creates a WiFi hotspot named SprinklerSetup.
  2. Connect to SprinklerSetup from your phone or laptop.
    Password: sprinkler123
  3. A setup page opens automatically. If it doesn't, open a browser and go to 192.168.4.1.
  4. Tap Configure WiFi, select your home network, enter the password, and tap Save.
  5. The board reboots and connects to your network.
  6. Open a browser on any device connected to the same WiFi and go to:
    http://sprinkler.local
    The SprinKlr-8 interface will load. No IP address needed.
    Note: sprinkler.local works automatically on iPhone, iPad, Mac, Android, and Windows 10/11. If it doesn't load, check that your device is on the same WiFi network as the controller.
Section 5

Using the Web Interface

Open a browser on any device on the same WiFi network and go to http://[board IP]/. The interface works on phones, tablets, and desktops.

Dashboard

Shows the current run status — which zone is active and how much time remains. Each zone has a Run and Stop button for quick manual control.

Schedule

Set your automatic watering schedule. Enable individual days, set a start time, and enter how many minutes each zone should run. The master toggle at the top enables or disables automatic scheduling without losing your settings.

Manual

Run a single zone for a specific number of minutes, or trigger a full day's schedule to run immediately. Useful for testing after installation.

Settings

View system information, rename zones, and apply over-the-air updates.

Section 6

Switching to a Different WiFi Network

To move the board to a new location or change your WiFi network, use the IO0 button on the board to clear the saved credentials.

Figure 4 — WiFi reset sequence. Releasing IO0 before 3 seconds cancels the reset.

After the reset, follow the First Boot & WiFi Setup steps in Section 4 to connect to the new network.

💡
No serial connection neededThis procedure works entirely with the buttons on the board. No laptop or cables required.
Section 7

Over-the-Air Updates

Once the board is on your network, all future updates are done through the web interface. No cables needed.

Update the Web Interface

  1. Open http://[board IP]/ in a browser.
  2. Go to Settings and scroll to the Updates section.
  3. Tap Download & Apply from GitHub.
  4. The board downloads the latest interface and the page reloads automatically.

Update the Firmware

  1. Open http://[board IP]/ in a browser.
  2. Go to Settings and scroll to the Updates section.
  3. Tap Download & Flash from GitHub and confirm the prompt.
  4. The board downloads and installs the new firmware, then reboots. Wait approximately 15 seconds before refreshing the page.
⚠️
During a firmware updateAll relay zones are turned off automatically before flashing begins. Do not power off the board during an update.

Current Version

The current firmware version is shown next to the Firmware button in the Settings → Updates section.

Section 8

Troubleshooting

ProblemSolution
Can't find board on network Confirm board is powered on and status LED is blinking. Try http://sprinkler.local from a device on the same WiFi. As a fallback, check your router's device list for sprinkler to get the IP address directly.
Web page won't load Confirm your phone/laptop is on the same WiFi as the board. Try the IP address directly instead of http://sprinkler/.
Zone doesn't activate Check wiring at the relay terminal for that zone. Use Manual → run zone and listen for the valve to click. Confirm the Common wire is connected.
Schedule not running Check the master schedule toggle in the Schedule page is ON. Confirm the correct days are enabled and start time is set.
LittleFS upload fails (error code 2) Close the Serial Monitor first — it blocks the COM port. Then retry with the IO0/EN boot sequence.
OTA firmware update fails Confirm the board has internet access. Check that a release exists on GitHub with a file named sprinkler_controller.bin.
Board won't enter flash mode Make sure to hold IO0 before tapping EN, and release IO0 only after "Connecting..." appears in the IDE.
SprinklerSetup hotspot not appearing Perform the WiFi reset procedure in Section 6 to force the setup portal to open.
