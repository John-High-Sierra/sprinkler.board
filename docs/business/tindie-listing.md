## Description

A smart 8-zone sprinkler controller built on ESP32. Runs entirely on your local
network — no cloud account, no app, no subscription, no data tracking.
Control it from any browser on any device.

**WORKS WITH HOME ASSISTANT — local API, no cloud relay required.**

### What's included
- Assembled SprinKlr-8 board in ABS enclosure
- Quick start card
- Source-available firmware (MIT + Commons Clause)

### Features
- 8-zone SPST relay outputs for 24VAC irrigation valves
- WiFi setup via captive portal — no config file editing, no app
- Mobile-first web interface: Dashboard, Schedule, Manual, Settings
- Per-zone scheduling with 7-day calendar
- OTA firmware updates via web UI (pulled from GitHub)
- USB-C programming port for firmware development
- Fully local — works without internet once configured
- No cloud account required. Ever.

### Why SprinKlr-8?

Most smart sprinkler controllers require a cloud account to function. If the
company's servers go down, your schedule stops. If they discontinue the product,
your controller becomes a brick. If your internet drops on a hot day, your
lawn dies.

SprinKlr-8 runs entirely on your local network. The web interface loads from
the board itself. Schedules run from the board's internal clock. No internet
required after initial WiFi setup.

### Compatibility
- Works with any standard 24VAC irrigation valve solenoid
- Compatible with Home Assistant via local REST API (no cloud relay)
- Any browser, any device — no app download
