SprinKlr-8
Wi-Fi Sprinkler Controller  ·  First-Time Setup
⚠️
Chrome or Edge required. This page uses the Web Serial API which only works in Google Chrome and Microsoft Edge (desktop). Please open this page in one of those browsers.
1

Connect the board to your computer

Plug the SprinKlr-8 into a USB port using a data-capable USB-C cable (charge-only cables won't work). The board has a CH340C USB-to-serial chip — Windows usually installs the driver automatically. If not, download the CH340 driver here.

Once plugged in, the green power LED on the board should light up.

2

Install the firmware

Click the button below to flash the SprinKlr-8 firmware directly from your browser. Chrome will ask you to select a serial port — choose the one labelled CH340 or USB Serial.

Web Serial is not supported in this browser. Please use Chrome or Edge on a desktop computer. This will erase any existing firmware and write the latest v1.3.0 release.
What happens next: After flashing the board will restart. Click ▶ View Boot Log below — the board will print step-by-step setup instructions directly in the log, including the Wi-Fi network name and what to do next.
📟 Boot Log
Click "▶ View Boot Log" after flashing to see setup instructions from the board...
3

Enter your Wi-Fi credentials

A dialog box will appear asking for your Wi-Fi network (SSID) and password. Enter them and click Connect. The board will connect and report its address.

If the dialog doesn't appear automatically, the board is still reachable via its setup hotspot — see Troubleshooting below.

4

Open the sprinkler dashboard

Once connected, click the link shown in the dialog, or open your browser and go to:

http://sprinkler.local

If sprinkler.local doesn't work on your network, use the IP address shown in the Wi-Fi dialog instead.

You can now unplug the USB cable — the board runs on its 24 VAC sprinkler power supply from this point on.

5

Set your timezone

In the dashboard, tap the ⚙ Settings tab and select your timezone from the dropdown. Tap Save. The clock in the header will update immediately, and schedules will run at the correct local times.

Troubleshooting

The Install button says "Unsupported"

Web Serial only works in Google Chrome and Microsoft Edge on a desktop or laptop computer. It does not work on phones, tablets, Firefox, or Safari. Open this page in Chrome or Edge.

No serial port appears in the list

Check that you're using a data cable (not a charge-only cable). Install the CH340 driver on Windows if the port doesn't appear. On Mac, the port shows as /dev/cu.usbserial-* — no driver needed on macOS 12+.

The Wi-Fi dialog didn't appear after flashing

This is a known Chrome/ESP32 timing quirk. Click Logs & Console in the installer, wait a few seconds for the boot messages to scroll by, then click Back. The "Configure Wi-Fi" button will appear.

If the 30-second window has already passed, configure Wi-Fi the classic way: connect your phone or laptop to the SprinklerSetup Wi-Fi (password: sprinkler123) — a setup page opens automatically.

sprinkler.local doesn't open

mDNS (the technology behind .local addresses) can be blocked on some corporate or guest networks. Find the board's IP address by logging into your router's admin page and looking for a device named sprinkler, then open that IP directly in your browser.

I need to change the Wi-Fi network

Power on the board and immediately hold the IO0 / BOOT button for 3 seconds. The board will erase its saved Wi-Fi credentials and open the SprinklerSetup hotspot again so you can enter new ones.

Firmware update failed partway through

If the flash was interrupted, the board may not boot. Simply click Install SprinKlr-8 Firmware again — it will retry the full flash from scratch and recover the board.
